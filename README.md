# Notepad

## <b>作品名稱:</b> 簡單的基本文字編輯器
## <b>程式語言:</b>C++
## <b>程式介紹:</b>
### 一個簡單的文字編輯器，包含複製貼上、增加新檔、存檔、上一步、下一步、更換背景、字體顏色。
## <b>程式使用方式:</b>
### 執行該程式，程式出現一個視窗，視窗最上面有圖示可以選擇你需要的操作，包含複製貼上、開啟檔案、增加新檔、存檔、上一步、下一步、更換背景、字體和字體顏色。
